## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/RAJCHITTORIYA/React-Playground.git
```

Go to the project directory

```bash
cd React-Playground
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
